Διασκεδάστε με τις φωτογραφίες σας

Το Shotwell είναι ένας εύχρηστος διαχειριστής φωτογραφιών, έτοιμος για όλα σας τα gadget. Συνδέστε την κάμερα ή το τηλέφωνό σας για να μεταφέρετε τις φωτογραφίες σας. Έπειτα, είναι εύκολο να τις μοιραστείτε και να τις κρατήσετε ασφαλείς. Εάν έχετε δημιουργική διάθεση, μπορείτε να δοκιμάσετε πολλές από τις εφαρμογές φωτογραφίας που υπάρχουν στο Κέντρο λογισμικού.
